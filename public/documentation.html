This is an application that will let you sort through different genres of movies
and directors, and let you save your favorite movies to your account.

Business Logic URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Allows a user to be created /users POST None A JSON object of the new user, structured like: {"Username": "", "id": "", "favoriteMovie": ""} A JSON object of the new user added, structured like: {"Username": "", "id": "", "favoriteMovie": ""}
Allows users to change their username /users/:id PUT :id None A JSON object of the user
Allows users to add a movie to their favorites /users/:id/:movieTitle POST :id, :movieTitle None Text confirming the movie has been added to the user's list
Allows users to remove a movie from their favorites /users/:id/:movieTitle DELETE :id, :movieTitle None Text confirming the movie has been removed from the user's list
Allows a user to be deleted /users/:id DELETE :id None Text confirming the user has been deleted
Returns a list of all movies /movies GET None None A JSON object of all the movies
Returns data about single movie by title /movies/:title GET :title None A JSON object of a single movie
Returns data about a genre by name /movies/genres/:genreName GET :genreName None A JSON object of a specific genre
Returns data about a director by name /movies/directors/:name GET :directorName None A JSON object of a specific director
